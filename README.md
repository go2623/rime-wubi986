986五笔鼠须管配置
# 使用方法
1. 点击Mac右上角状态栏中鼠须管图标
2. 点击【用户设定...】，此时会弹出一个文件夹窗口，名称为Rime
3. 备份后，清空Rime文件夹的内容
4. 导入该项目中的配置文件
5. 点击Mac右上角状态栏中鼠须管图标，点击【重新部署】，就可以愉快地用986五笔方案打字了。
# 功能备注
1. 遇到不会打的字，点击~键，可以输入拼音，反查这个字的986五笔编码。
2. Z键重复输入
3. 默认取消四码唯一自动上屏，因为对词库不熟悉会多打空格，可以去wubi986.schema.yaml文件第54行附近修改。
4. 默认开启编码逐级提示，可在wubi986.schema.yaml文件第59行附近修改.
